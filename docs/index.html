Why MailMind

📬

Multi-Mailbox Support

Works natively with 126, 163, QQ Mail, Gmail (OAuth & App Password), and Outlook (OAuth). IMAP IDLE for instant push delivery.

🤖

10+ AI Backends

Plug in any AI — Claude Code, Codex, Gemini CLI (local CLI mode), or call Anthropic, OpenAI, Gemini, Qwen, DeepSeek, Copilot APIs directly.

⏰

Scheduled Tasks

Natural language scheduling. "Every day 8am weather Tokyo" or "Every Monday news AI summary". Persisted across restarts.

🔒

Whitelist Security

Per-mailbox sender whitelists. Only trusted addresses or entire domains can trigger AI processing. Strangers are silently ignored.

📎

Attachment Support

Attach .txt, .md, .csv files to enrich AI context. AI can return file attachments in replies too.

🛠️

One-Script Management

manage.sh start|stop|restart|log|status — plus systemd install for auto-start on boot. Setup wizard included.

🌐

Web Search Injection

Optionally inject live web results into AI context. DuckDuckGo, Bing, Google, or Wikipedia — your choice per query.

📊

Report Archiving

Scheduled tasks can archive results to reports/ as timestamped files — ideal for daily digests and monitoring.

The Process

Three steps.
Zero friction.

01

Configure once

Run bash manage.sh setup to launch the interactive wizard. Pick your mailbox, choose an AI backend, set credentials. Done in minutes.

02

Start the daemon

Run bash manage.sh start. MailMind connects via IMAP IDLE (or poll mode) and waits for your emails in the background.

03

Email your instructions

Send any instruction as an email body. The AI processes it and replies directly to your inbox — with an AI-crafted subject line.

Scheduled Tasks

Automate your
information diet

Write natural-language schedules in your email. MailMind parses time expressions in Chinese, Japanese, Korean, and English — and runs tasks automatically.

Results are emailed to you and/or archived to reports/ as timestamped files.

weather news web_search ai_job report system_status email
example instructions
# Daily morning weather
Every day 08:00 weather Tokyo # News digest every 2 hours
Every 2 hours AI news summary, archive # Monday briefing with AI
Every Monday 10:00 AI job: summarize
  last week and plan this week, archive # Server health every 10 min
Every 10 minutes system status ✓ Tasks scheduled and persisted

Quick Start

Up in three
commands

1

Clone the repo

Download MailMind from GitHub and enter the project directory.

2

Run setup wizard

bash manage.sh setup — creates a virtualenv, installs dependencies, and walks you through configuration interactively.

3

Start the daemon

bash manage.sh start — runs MailMind in the background. Send yourself an email and get an AI reply within 60 seconds.

4

Optional: systemd

bash manage.sh install — register as a systemd service for auto-start on boot. Includes status, log, and restart commands.

bash · terminal
$ git clone https://github.com/
  riangone/MailMindAgent.git $ cd MailMindAgent $ bash manage.sh setup
  → Creating virtualenv...
  → Installing dependencies...
  → Configuring .env...
  ✓ Setup complete! $ bash manage.sh start

  ✓ Listening on gmail via IMAP IDLE
